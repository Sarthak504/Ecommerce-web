# 🛍️ OtakuSip — Anime Mug Shopping Website

OtakuSip is a **modern, mobile-first e-commerce frontend** built using pure HTML, CSS, and JavaScript.
It enables small businesses to **collect and manage customer orders online without requiring a backend or payment gateway**.

---

## 🔗 Live Demo

👉 https://sarthak504.github.io/Ecommerce-web/

---

## 📌 Overview

This project simulates a lightweight e-commerce platform focused on **order collection rather than online payments**.
It is designed specifically for small businesses that want to establish an online presence quickly with **zero infrastructure cost**.

---

## ✨ Key Features

* **Product Catalog**

  * Organized product listings with category-based filtering
  * Clean and responsive grid layout

* **Shopping Cart System**

  * Add to cart with quantity management
  * Increment / decrement / remove items
  * Persistent cart using browser localStorage

* **Checkout & Order Handling**

  * Customer details form (name, phone, address)
  * Automated order summary generation
  * Email-based order submission via Formspree

* **User Experience**

  * Mobile-first responsive design
  * Hero section with anime-themed branding
  * Smooth interactions and intuitive UI

* **Business Integration**

  * WhatsApp quick contact for order confirmation
  * No payment gateway required

---

## ⚙️ System Workflow

1. User browses available products
2. Items are added to cart
3. Cart state is stored locally using localStorage
4. User fills checkout form
5. Order details are compiled dynamically
6. Formspree sends order summary via email
7. Business contacts customer via WhatsApp

* **Fast Load Times** — Minimal assets, CDN-based images
